# TaskFlow

TaskFlow is a SaaS-oriented task management application designed with a scalable full-stack architecture.
It provides secure authentication, protected user access, and RESTful APIs suitable for multi-user, cloud-deployed environments.
The application follows modern SaaS engineering practices, including token-based authentication, separation of frontend and backend concerns, and extensible API design.

## SaaS Features
- JWT-based authentication (access & refresh tokens)
- Protected frontend routes and role-secured APIs
- Scalable REST architecture for multi-user usage
- Pagination, filtering, and ordering for large datasets
- Decoupled frontend and backend for independent scaling
- Cloud-deployment ready structure
